# Get Started with Tazapay

To get started with Tazapay, you will need to create an account on the [sandbox (test) environment](https://dashboard-sandbox.tazapay.com).

# Access your API keys

Once the sandbox account is active, fetch your API keys to call Tazapay's endpoints.

- Log in to the Tazapay Sandbox Dashboard.
- On the left hand panel, select Settings.
- Navigate to Integration -> API Keys and Developer Docs.
- You should be able to view the keys.
