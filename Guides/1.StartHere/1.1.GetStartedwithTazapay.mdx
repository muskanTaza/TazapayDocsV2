To get started with Tazapay, you will need to create an account on the [<u>sandbox (test) environment</u>](https://dashboard-sandbox.tazapay.com/signup).

![Screenshot2025 12 14at10 35 00PM Pn](/images/Screenshot2025-12-14at10.35.00PM.png)

## \\

**Access your API keys**

Once the sandbox account is active, fetch your API keys to call Tazapay's endpoints.

1. Log in to the [<u>Tazapay Sandbox Dashboard</u>](https://dashboard-sandbox.tazapay.com/).
2. On the left hand panel, select Settings.
3. Navigate to Integration -\> API Keys and Developer Docs.
4. You should be able to view the keys.
