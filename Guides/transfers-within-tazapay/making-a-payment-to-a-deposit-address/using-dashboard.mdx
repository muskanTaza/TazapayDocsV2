Follow the steps below to create a payout using a deposit address:

1. Navigate to **Create Payout** in the dashboard.
2. If the beneficiary does not exist, add the beneficiary inline.
3. Enter the beneficiaryâ€™s **Deposit Address**.
4. Enter the payout details:
   - **Amount**
   - **Source balance currency**
   - **Payment reason**
   - **Description**
5. Confirm the details and submit the payout.

---

## Key Fields Explained

| Field | Description |
| --- | --- |
| **Beneficiary / Deposit Address** | The Tazapay deposit address where the funds will be sent |
| **Amount** | The amount to be transferred |
| **Balance Currency** | The currency from which the balance will be debited |
| **Reason for Payment** | Optional field to describe the purpose of the payout |
| **Description** | Narrative for internal bookkeeping |
