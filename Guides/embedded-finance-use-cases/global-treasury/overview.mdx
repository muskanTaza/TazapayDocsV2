Tazapay allows businesses to:

<CardGroup cols={2}>

  </Card>
  <Card title="Hold Multi-Currency Balances" icon="wallet">
    Maintain balances across multiple currencies in one place
  </Card>
  <Card title="Currency Conversion" icon="arrows-rotate">
    Convert between currencies when required at competitive rates
  </Card>
  <Card title="Global Payouts" icon="globe">
    Distribute funds worldwide using local payment rails
  </Card>

| Capability | Description |

| Balances | Multi-currency holding |
| Conversion | On-demand FX |
| Payouts | Local rails across countries |
