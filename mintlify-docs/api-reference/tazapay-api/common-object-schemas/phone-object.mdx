title: "Phone Object"

## Object Structure

```json Phone
{
  "calling_code": "91",
  "number": "9231231231"
}
```

  

## Object Parameters

| Field | Type | Description |
| :--- | :--- | :--- |
| calling\_code | string | The international calling code for the customer's phone number. |
| number | string | The phone number of the customer. |
