# Changelog

Stay up to date with the latest changes, improvements, and updates to Tazapay's platform and features.

## December 2025

### December 28, 2025

**Improvements**
- Enhanced payout processing speed by 40% for SEPA transfers
- Added support for real-time balance updates across all currencies
- Improved webhook reliability with automatic retry mechanism

**Bug Fixes**
- Fixed issue with FX rate locking during maker-checker approval flow
- Resolved display issue for multi-currency balances on mobile dashboard
- Corrected timezone handling for transaction timestamps in SGT region

### December 15, 2025

**New Features**
- Introduced Lightning Network support for Bitcoin payouts
- Added bulk payout scheduling functionality
- Launched new balance transaction export feature (CSV/PDF)

**Improvements**
- Optimized API response times by 25%
- Enhanced error messages for failed beneficiary verifications

## November 2025

### November 30, 2025

- Improved transaction search with advanced filters

### November 12, 2025

**Bug Fixes**
