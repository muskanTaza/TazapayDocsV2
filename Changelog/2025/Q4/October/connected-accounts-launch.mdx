# Connected Accounts Launch

**Date: October 25, 2025**

## New Features

- Launched connected accounts feature for platforms
- Added support for INR local payouts via UPI
- Introduced customizable webhook event subscriptions

## Improvements

- Reduced payout settlement time for local transfers
- Enhanced beneficiary validation for bank accounts
- Improved API documentation with interactive examples
