# October 2025

## October 25, 2025

### New Features
- Launched connected accounts feature for platforms
- Added support for INR local payouts via UPI
- Introduced customizable webhook event subscriptions

### Improvements
- Reduced payout settlement time for local transfers
- Enhanced beneficiary validation for bank accounts
- Improved API documentation with interactive examples

## October 8, 2025

### Improvements
- Updated currency conversion rates to refresh every 30 seconds
- Enhanced transaction history with detailed status descriptions
- Improved email notifications with better formatting

### Bug Fixes
- Fixed duplicate webhook delivery issue
- Resolved IBAN validation error for certain EU banks
- Corrected display issue in transaction fee breakdown
