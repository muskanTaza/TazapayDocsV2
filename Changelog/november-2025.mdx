# November 2025

## November 30, 2025

### New Features
- Released stablecoin wallet support (USDC, USDT)
- Added virtual account auto-reconciliation
- Introduced new payout purpose codes for compliance

### Improvements
- Updated KYB document verification process
- Enhanced dashboard UI for better mobile experience
- Improved transaction search with advanced filters

## November 12, 2025

### Bug Fixes
- Fixed race condition in concurrent payout processing
- Resolved issue with entity status updates via webhook
- Corrected calculation error in cross-border FX fees

### Security
- Implemented additional 2FA options (authenticator apps)
- Enhanced API rate limiting for sensitive endpoints
- Updated TLS certificates to latest security standards
