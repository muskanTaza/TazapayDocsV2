# Endpoints and Environments

Tazapay provides two environments for API integration.

## Environments

### Sandbox Environment
- **Base URL:** `https://service-sandbox.tazapay.com`
- **Purpose:** Testing and development
- **Data:** Test data only, no real transactions

### Production Environment
- **Base URL:** `https://service.tazapay.com`
- **Purpose:** Live transactions
- **Data:** Real transactions and payments

## API Versioning

All API endpoints are versioned. The current version is `v3`.

Example endpoint: `/v3/payout`

## Regional Availability

Tazapay API is available globally with optimized routing for:
- Asia Pacific
- Europe
- Americas
