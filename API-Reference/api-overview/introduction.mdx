# API Introduction

Welcome to the Tazapay API documentation. This API allows you to integrate Tazapay's payment infrastructure into your application.

## Getting Started

To get started with the Tazapay API, you'll need:

1. A Tazapay account
2. API credentials (API Key and Secret)
3. Understanding of RESTful APIs

## Base URLs

- **Sandbox:** `https://service-sandbox.tazapay.com`
- **Production:** `https://service.tazapay.com`

## API Features

- Payment processing
- Entity management
- Payout capabilities
- Balance inquiries
- Transaction tracking
