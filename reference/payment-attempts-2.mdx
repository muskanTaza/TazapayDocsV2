---
title: "Payment Attempts"
---

| ERROR CODE | Meaning                                                                                               |
| ---------- | ----------------------------------------------------------------------------------------------------- |
| 15500      | Checkout ID field is required and must be a valid ID, Example: chk\_id                                |
| 15515      | Request object passed for this API request is failing. Please provide valid metadata                  |
| 15516      | Escrow ID field is required and must be a valid ID, Example: esc\_id                                  |
| 15550      | Payin Id is required. Please check whether id provided is in correct format. Example: chk\_id/esc\_id |
| 19100      | Invalid payment*attempt\_id, should have prefix 'pat*' and should be followed by valid xid            |
