# How It Works

* A business or individual uses their online or mobile banking platform to send funds directly to another person's or business's bank account.
* This typically requires the recipient's bank account details (account number and bank name).
* The sender authenticates the transaction through their bank's security protocols.
* Funds are then transferred directly from the sender's bank account to the recipient's bank account.

# Payment Method type

Users can use the below payment method type to create a transaction using Bank Push in the available geography

* Vietnam: `bank_push_vnd`

# Limits and Expiration

The following is the upper and lower limits for Bank Push transactions:

* **Minimum Limit**: USD 0.5
* **Maximum Limit**: USD 19000

The default expiration for a transaction is **30 Days.**
