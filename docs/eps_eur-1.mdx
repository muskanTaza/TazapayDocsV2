# How It Works

* Customers are redirected to their online banking portal.
* After logging in, they review and authorize the payment, which is processed instantly.

# Payment Method type

Users can use the below payment method type to create a transaction using EPS in the available geography

* Austria: `eps_aus`

# Functionality

| Functionality           | Yes/No | Description                                               |
| ----------------------- | ------ | --------------------------------------------------------- |
| **Refund Support**      |        |                                                           |
| Full Refund             | ✅      |                                                           |
| Partial Refund          | ✅      |                                                           |
| **Generally Available** | ❌      | Requires onboarding approval which takes about 7-10 days. |
| Dispute and Chargebacks | ✅      |                                                           |

# Limits and Expiration

The following is the upper and lower limits for EPS transactions:

* **Minimum Limit**: USD 1
* **Maximum Limit**: USD 15000

The default expiration for a transaction is **1 hour.**
