# How It Works

* Users link their Chinese bank accounts to their WeChat app.
* Payments are typically made by scanning a QR code displayed by the merchant or by presenting their own QR code for the merchant to scan.

# Payment Method type

Users can use the below payment method type to create a transaction using WeChat Pay in the available geography

* Thailand: `wechat_thb`

# Functionality

| Functionality               | Yes/No | Description                                                                                                    |
| --------------------------- | ------ | -------------------------------------------------------------------------------------------------------------- |
| **Refund Support**          |        |                                                                                                                |
| Full Refund                 | ✅      | Made to phone number if provided, else with account details                                                    |
| Partial Refund              | ✅      | Made to phone number if provided, else with account details                                                    |
| **Generally Available**     | ✅      | Means that the payment method is available by default and does not need any additional configuration/approval. |
| **Dispute and Chargebacks** | ❌      |                                                                                                                |

# Limits and Expiration

The following is the upper and lower limits for WeChat Pay transactions:

* **Minimum Limit**: USD 0.02
* **Maximum Limit**: USD 200000

The default expiration for a transaction is **2 hours.**
