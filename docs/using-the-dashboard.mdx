![Step 2 screenshot](/images/docs/workflows/92cff27a-339d-4519-b744-ae48e5919b57/steps/9fd856ad-bc1e-4e73-8370-82a75680fc1b/28f1dc14-59f3-40fe-bc9a-a98761a3863d.png)

### 3. Click on Tazapay Account

![Step 3 screenshot](/images/docs/workflows/92cff27a-339d-4519-b744-ae48e5919b57/steps/b794aece-a438-40d6-9a7f-2e2cefbd1fcd/26a72743-d3e2-4d9a-8283-fd6c72b3b85b.png)

### 4. Type the deposit address of the beneficiary

![Step 4 screenshot](/images/docs/workflows/92cff27a-339d-4519-b744-ae48e5919b57/steps/ddd38450-8b30-4071-9580-47963449f331/42457ca4-b64c-4a63-ae92-1ac5ed84ae8a.png)

### 5. Enter the payout amount

![Step 5 screenshot](/images/docs/workflows/92cff27a-339d-4519-b744-ae48e5919b57/steps/7f21bb59-6468-4682-b37c-f2e2d25835ef/0149376e-8918-4249-88cf-33ac8893b2b9.png)

### 6. Select the balance currency you want to fund the payout from

![Step 6 screenshot](/images/docs/workflows/92cff27a-339d-4519-b744-ae48e5919b57/steps/9a00b9c7-c03e-46db-a829-c01f63aacce5/5652bbd4-8eb1-4d4e-b46a-8757c09bed44.png)

### 7. Select the reason for payment

![Step 7 screenshot](/images/docs/workflows/92cff27a-339d-4519-b744-ae48e5919b57/steps/95650457-56a2-4816-a483-103da2dedf9f/14e5acb1-104a-458c-bbe9-f0cd2d6e74e1.png)

Type "This is a payment for payment of invoice number 4298472824"
