# How it Works?

* Customers access their bank accounts through the bank's website or a dedicated mobile application using secure credentials. They can perform various functions such as:

  * **Account Management:** View balances, transaction history, and download statements.
  * **Fund Transfers:** Transfer money between their accounts and to third-party accounts within the same or different banks.
  * **Bill Payments:** Pay utility bills, credit card bills, and other invoices online.
  * **Mobile Top-ups:** Recharge prepaid mobile phone credit.
  * **Loan and Investment Management:** View loan details, make payments, and manage investments.
  * **Other Services:** Request chequebooks, update personal information, and communicate with the bank.

# Limits and Expiration

The following is the upper and lower limits for Internet Banking transactions:

* **Minimum Limit**: USD 0.5
* **Maximum Limit**: USD 100k

The default expiration for a transaction is **15 minutes.**
