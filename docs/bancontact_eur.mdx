# How It Works

* Users have a Bancontact card linked to their Belgian bank account.
* For online payments, they are redirected to their bank's secure environment to authorize the transaction.
* For in-store payments, they use their Bancontact card at POS terminals.

# Payment Method type

Users can use the below payment method type to create a transaction using Bancontact in the available geography

* Belgium: `bancontact_eur`

# Functionality

| Functionality           | Yes/No | Description                                               |

| **Refund Support**      |        |                                                           |

| Partial Refund          | ✅      |                                                           |
| **Generally Available** | ❌      | Requires onboarding approval which takes about 7-10 days. |
| Dispute and Chargebacks | ✅      |                                                           |

# Limits and Expiration

The following is the upper and lower limits for Bancontact transactions:

* **Minimum Limit**: USD 0.01
* **Maximum Limit**: USD 15000

The default expiration for a transaction is **1 hour.**
