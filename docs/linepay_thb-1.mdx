# How it Works?

* Users link their bank accounts or credit cards to their Line Pay accounts.
* Payments are made by scanning QR codes at physical stores or using the app for online transactions.
* It also facilitates peer-to-peer transfers.

# Payment Method type

Users can use the below payment method type to create a transaction using Line Pay in the available geography

* Thailand: `linepay_thb`

# Functionality

| Functionality                | Yes/No | Description                                                                                                    |
| ---------------------------- | ------ | -------------------------------------------------------------------------------------------------------------- |
| **Refund Support**           |        |                                                                                                                |
| Full Refund                  | ✅      | Made to phone number if provided, else with account details                                                    |
| Partial Refund               | ✅      | Made to phone number if provided, else with account details                                                    |
| **Generally Available**      | ✅      | Means that the payment method is available by default and does not need any additional configuration/approval. |
| **Disputes and Chargebacks** | ❌      |                                                                                                                |

# Limits and Expiration

The following is the upper and lower limits for Line Pay transactions:

* **Minimum Limit**: USD 0.02
* **Maximum Limit**: USD 1300

The default expiration for a transaction is **20 minutes.**
