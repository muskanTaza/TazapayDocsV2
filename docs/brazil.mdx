# How it Works?

* Users link their bank accounts or digital wallets to PIX.
* Payments are made instantly, 24/7, including weekends and holidays.
* Users can pay by scanning a QR code, using a PIX key (email, phone number, CPF/CNPJ), or manually entering bank details.

# Payment Method type

Users can use the below payment method type to create a transaction using PIX in the available geography

* Brazil: `pix_brl`

# Limits and Expiration

The following is the upper and lower limits for PIX transactions:

* **Minimum Limit**: USD 0.01
* **Maximum Limit**: USD 20000

The default expiration for a transaction is **20 minutes.** This is further configurable within the default time of expiration.
