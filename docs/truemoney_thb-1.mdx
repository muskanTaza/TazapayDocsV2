# How It Works

* Users can top up their TrueMoney wallets through various channels like bank transfers, convenience stores, or cash deposits.
* Payments are made by scanning QR codes, using the app for online transactions, or transferring funds to other users.

# Payment Method type

Users can use the below payment method type to create a transaction using TrueMoney in the available geography

* Thailand: `truemoney_thb`

# Functionality

| Functionality               | Yes/No | Description                                                                                                    |
| --------------------------- | ------ | -------------------------------------------------------------------------------------------------------------- |
| **Refund Support**          |        |                                                                                                                |
| Full Refund                 | ✅      | Made to phone number if provided, else with account details                                                    |

| **Generally Available**     | ✅      | Means that the payment method is available by default and does not need any additional configuration/approval. |

# Limits and Expiration

The following is the upper and lower limits for TrueMoney transactions:

* **Minimum Limit**: USD 0.03
* **Maximum Limit**: USD 1300

The default expiration for a transaction is **2 hours.**
