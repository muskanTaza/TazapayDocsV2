A popular mobile wallet and payment service offered by Vietnam Posts and Telecommunications Group (VNPT).

# How It Works

* Users link their bank accounts or top up their VNPT Pay wallets.
* Payments are made by scanning QR codes, making online transactions, or transferring funds to other users.

# Payment Method type

Users can use the below payment method type to create a transaction using VNPTPay in the available geography

* Vietnam: **`vnptpay_vnd`**

# Functionality

| Functionality               | Yes/No | Description                                                                                                    |
| --------------------------- | ------ | -------------------------------------------------------------------------------------------------------------- |
| **Refund Support**          |        |                                                                                                                |
| Full Refund                 | ✅      |                                                                                                                |
| Partial Refund              | ❌      |                                                                                                                |
| **Generally Available**     | ✅      | Means that the payment method is available by default and does not need any additional configuration/approval. |
| **Dispute and Chargebacks** | ❌      |                                                                                                                |

# Limits and Expiration

The following is the upper and lower limits for VNPTPay transactions:

* **Minimum Limit**: USD 0.07
* **Maximum Limit**: USD 1800

The default expiration for a transaction is **10 minutes.**
