# Images Directory

Store all documentation images here.

## Structure:
- `/screenshots` - UI screenshots
- `/diagrams` - Architecture and flow diagrams
- `/logos` - Brand assets and logos
- `/icons` - Icon assets

## Naming Convention:
- Use kebab-case: `payment-flow-diagram.png`
- Be descriptive: `dashboard-analytics-view.png`
- Include version if needed: `api-v3-architecture.png`

## Supported Formats:
- PNG (recommended for screenshots)
- JPG (for photos)
- SVG (for diagrams and icons)
- GIF (for animations)

## Usage in MDX:
```mdx
![Alt text](/images/your-image.png)
```
